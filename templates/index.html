What is Singapore Gaming Syndicate?

Singapore Gaming Syndicate (SGS) is Singapore's biggest verified Mobile Legends community. We run daily scrims, publish this live hero guide, and help local MLBB players find teammates at every rank — from Epic through Mythic Glory.

Every hero page on this site is generated from real ranked match data: live win rate, pick rate, ban rate, counter matchups, synergy partners, skill combos, and the community's highest-voted builds. Stats refresh every 6 hours so you're never reading a stale guide. If you want to practise what you read, join the SGS Telegram group — free, no sign-up, verified members only.

About This Data

Source of Truth

Every win-rate, pick-rate and ban-rate figure on this site is pulled from the OpenMLBB API — a public aggregation of Mobile Legends ranked match data across all tiers from Epic to Mythic Glory. We do not manually curate tier lists. Rankings emerge from real player behaviour in the last 7 days.

Update Frequency

Upstream stats refresh every 6 hours, with a 12-hour graceful fallback if the API is unreachable. Every page shows a "Last updated" badge with an exact timestamp — always check it. If the badge shows more than 12 hours old, the data is from our cold cache and the live API is likely down.

Why Trust Us?

SGS is maintained by active Mythic-rank Singapore players. All build recommendations, skill-order advice and 3-hero pool calls are reviewed against the live data and the Mobile Legends pro scene (MPL-SG, MSC). No AI-hallucinated builds, no affiliate-driven hero rankings, no hype cycles — just the numbers and the experience to interpret them.

Methodology

  • Tier split: SS ≥54% WR, S ≥52%, A ≥50%, B ≥48%, C ≥46%, D <46%.
  • Counter strength: Hard +3% WR boost, Soft +1-3%, Minor <1%.
  • 3-hero pool: comfort (highest pick), counter (high WR + low pick), safe blind (highest WR in top-5 most-picked).
  • Trending: sorted by current ban rate — the cleanest proxy for perceived strength.
